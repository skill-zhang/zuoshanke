核心理念

⛰️

本体

坐山客是系统真正的主人。本体不朽、不衰减，是观察一切的永恒之眼。

🫧

分身

每个场景/频道是一个独立分身。分身不知本体事，prompt 层完全隔离。

🧠

双重记忆

本体记忆不朽不衰减，分身记忆正常衰减。缓存全量存储，注入层自适应取 Top-N。

🔄

Agent Loop

LLM 自主决策驱动的执行引擎。工具调用、收敛检测、子 Agent 协作，全链路自主闭环。

核心能力

Thinking Map

Browser Dial Test

设计原则

  • 01

    数字原生

    输出化为网页 / Markdown，而非建议打印。数字世界拥有数字世界的表达方式。

  • 02

    LLM 自主决策

    不造脆弱规则。LLM 翻车多因 context 不充分，非自主之过。给够上下文，让 AI 自己决定。

  • 03

    开放 > 封闭

    坐山客开放生态。Avatar 是本体具象化的出口，不是装饰品。工具、模型、通道皆可扩展。

  • 04

    先方案再动手

    充分讨论形成设计文档后落地。方向明确后不逐级确认，信任执行者的判断。

  • 05

    收敛触发统一

    所有收敛检测统一通过 function calling，而非文本标记。量化规则仅用于纯数学场景。
