# 2-5岁亲子出行通用物资需求清单

> 适用场景：短途（1-3天）、中途（4-7天）、长途（7天+）家庭出行
> 目标年龄：2-5岁幼儿（学龄前阶段）
> 版本：v1.0

---

## 一、衣物类 Clothing

| 物品 | 短途(1-3天) | 中途(4-7天) | 长途(7天+) | 备注 |
|------|------------|------------|-----------|------|
| 上衣（件） | 3-4 | 5-7 | 7-10+ | 纯棉/速干，备1件厚外套 |
| 裤子（条） | 2-3 | 4-5 | 5-7+ | 松紧带款，易穿脱 |
| 内裤（条） | 3-4 | 5-7 | 7-10+ | 每天至少1条+备用 |
| 袜子（双） | 3-4 | 5-7 | 7-10+ | 薄厚各半 |
| 外套 | 1件 | 1-2件 | 2件 | 防风防水优先 |
| 睡衣 | 1套 | 1-2套 | 2套 | 按住宿天数/洗涤频率 |
| 鞋子 | 2双 | 2-3双 | 3双 | 1舒适运动+1凉/雨鞋 |
| 帽子 | 1顶 | 1-2顶 | 2顶 | 防晒+保暖各1 |
| 口水巾/围兜 | 2-3条 | 3-5条 | 5-7条 | 吃饭/流涎必备 |
| 泳衣 | - | 1套 | 1套 | 如有水上活动 |
### 衣物打包要点
- 分层原则：薄厚搭配，洋葱式穿脱（贴身吸汗到中层保暖到外层防风）
- 备用套装：随身包必带1整套干净衣物（含鞋袜），应对泼洒/弄脏
- 标识：所有外套/帽子贴姓名标签，防止遗失
- 鞋类：不要穿新鞋出行，提前2周磨合

## 三、医疗类 Medical

### 3.1 常备药箱（儿童版）
| 类别 | 药品 | 说明 |
|------|------|------|
| 退热 | 布洛芬混悬液 | 按体重计算剂量 |
| 退热 | 对乙酰氨基酚滴剂 | 交替使用遵医嘱 |
| 体温计 | 电子/耳温枪 | 水银禁带（易碎+安检） |
| 腹泻 | 口服补液盐III | 预防脱水最重要 |
| 腹泻 | 蒙脱石散 | 物理止泻，餐前服用 |
| 过敏 | 西替利嗪滴剂/氯雷他定 | 遵医嘱备用 |
| 过敏 | 炉甘石洗剂 | 蚊虫叮咬/皮疹止痒 |
| 外伤 | 碘伏棉签（独立包装） | 一次性消毒伤口 |
| 外伤 | 创可贴（卡通款） | 防水+普通各若干 |
| 外伤 | 无菌纱布+医用胶带 | 较大伤口包扎 |
| 鼻塞 | 生理盐水喷鼻剂 | 飞机起降/干燥环境 |

| 护臀 | 氧化锌软膏 | 尿布疹预防 |

### 3.2 用药原则
- 出行前确认孩子无发热/腹泻/皮疹等不适症状
- 所有药品标注体重对应剂量，出发前咨询儿科医生
- 避光阴凉储存，部分需冷藏（打开后的混悬液）
- 随身携带过敏记录卡（中英文）
- 提前查好目的地三甲医院/连锁药店位置

### 4.2 安全防护
| 物品 | 说明 |
|------|------|
| 防走失背包/牵引绳 | 人流密集区必用 |
| 儿童安全座椅 | 车载必备 |

| 儿童退热指示卡 | 打印体温-体重-剂量对照表 |

### 4.3 应急应对
| 场景 | 应对方案 |
|------|---------|
| 发热（38.5度以上） | 物理降温+口服退热药，持续2天以上就医 |
| 腹泻（每天3次以上） | 停奶/油/糖，口服补液盐，观察尿量 |
| 外伤出血 | 清水冲洗-碘伏消毒-纱布包扎-就医 |
| 蚊虫叮咬过敏 | 局部冷敷-炉甘石-抗组胺药 |
| 走失 | 原地等待+联系工作人员+手机定位 |
| 噎食 | 海姆立克法（幼儿改良版），家长应提前学习 |
| 飞机耳压痛 | 喂水/奶/零食做吞咽动作 |

### 4.4 交通场景附加
**飞机出行附加**
- 起飞/降落时喂水或零食（缓解耳压）
- 便携玩具/贴纸书（座舱内安静玩耍）
- 多备一套衣服在随身行李中

**高铁/火车附加**
- 带沿小桌板贴纸/磁力画板
- 儿童口罩（车厢空气流通一般）
- 垃圾袋+湿巾（处理食物残渣）

**自驾附加**
- 安全座椅必须
- 车窗帘/遮阳挡
- 每2小时休息20分钟活动
- 车内垃圾袋+呕吐袋（防晕车）

**酒店入住检查清单**
- 检查插座是否有保护
- 窗户/阳台是否安全锁闭
- 地面有无尖角/小物件
- 热水壶位置是否够高
- 床周围无缝隙（防卡住）
- 确认最近的医院/药店位置

## 六、亲子出行黄金原则
1. 宁多勿少：2-5岁孩子需求不可预测，衣物/尿裤+20%余量
2. 分层拿取：随身包放最急需品，大箱放换洗和备用
3. 平常心：孩子会脏会闹会病，视为旅行的一部分
4. 规律作息：尽量保留午睡/饭点/就寝节奏
5. 慢节奏：每天安排1-2个活动即可
6. 就地补给：目的地药店/超市都买得到
7. 安全第一：防走失+防意外是最高优先级
