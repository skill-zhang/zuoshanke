# Weather — 天气查询

## 概述

通过 wttr.in 免费 API 获取城市实时天气，无需 API Key。含内存缓存（60秒TTL）和异常回退。

## 数据源

**wttr.in（主数据源）** — 免费，无 API Key
- 支持中英文城市名
- 返回实时天气（current_condition）
- 10秒超时，失败时自动回退到内置静态数据

**回退数据：** 内置9城市静态数据（北京/上海/深圳/广州/成都/杭州/武汉/南京/重庆），仅用于保底，不代表真实天气。

## 返回格式

```python
{
    "city": str,       # 城市名
    "temp": str,       # 温度，如 "20°C"
    "desc": str,       # 天气描述，如 "晴朗"、"多云"
    "humidity": str,   # 湿度，如 "45%"
    "wind": str,       # 风向风速，如 "东风 3级"
}
```

API 失败时返回值包含 `_source: "fallback"` 字段。

## 使用场景

- 聊天机器人回复天气
- Agent 决策（如下雨则延迟室外任务）
- 快速原型和演示

## 注意事项

1. **wttr.in 依赖外网：** 中国大陆部分地区可能无法直连，会触发回退
2. **回退数据是静态的：** 不反映真实天气
3. **无异步支持：** 当前使用同步 HTTP 请求
4. **无历史/预报数据：** 只返回当前实时天气
5. **缓存粒度是城市级别：** 60秒 TTL
