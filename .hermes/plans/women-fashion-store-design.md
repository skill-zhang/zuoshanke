# 女装售卖系统 — 设计方案

## 背景
开发一个完整的单页HTML女装售卖系统，包含商品展示、分类筛选、详情弹窗等功能，深色主题、响应式设计。

## 设计目标
- **功能目标**：商品卡片展示、分类导航、按分类筛选、商品详情弹窗（大图/价格/尺码/颜色/库存）
- **非功能目标**：深色主题风格、响应式设计适配移动端、真实女装图片引用、流畅交互

## 数据模型

### 商品 (Product)
```javascript
{
  id: number,
  name: string,          // 商品名称
  category: string,      // 分类：连衣裙/上衣/裤装/裙装/外套/配饰
  price: number,         // 价格
  originalPrice: number, // 原价（用于显示折扣）
  description: string,   // 简短描述
  images: string[],      // 图片URL数组（从unsplash等真实图源引用）
  colors: string[],      // 可选颜色
  sizes: string[],       // 可选尺码
  stock: number,         // 库存数量
  rating: number,        // 评分 1-5
  sales: number,         // 已售数量
  tags: string[]         // 标签如"新品""热卖""折扣"
}
```

### 分类 (Category)
```javascript
{
  id: string,
  name: string,
  icon: string   // emoji图标
}
```

## 功能规划

### 1. 顶部导航区
- 品牌Logo + 店名
- 购物车图标（显示数量）
- 搜索框

### 2. 分类导航栏
- 水平滚动的分类标签（含图标）
- 当前选中高亮
- "全部"作为默认分类

### 3. 商品展示区
- 网格布局卡片（桌面4列/平板3列/手机2列）
- 每张卡片包含：商品图片、名称、价格（含原价划线）、评分、标签
- 悬停效果：图片缩放、阴影加深
- 点击卡片打开详情弹窗

### 4. 商品详情弹窗
- 模态框覆盖层
- 左侧：大图展示（可轮播）
- 右侧：商品名称、价格、评分、描述
- 颜色选择器（色块点击）
- 尺码选择器（按钮组）
- 库存状态显示
- 加入购物车按钮
- 关闭按钮

### 5. 购物车功能
- 侧边栏抽屉形式
- 商品列表（缩略图、名称、数量、单价）
- 数量加减
- 删除商品
- 总计金额
- 结算按钮

### 6. 响应式设计
- 桌面(≥1024px)：4列网格
- 平板(768-1023px)：3列网格
- 手机(<768px)：2列网格，导航折叠

## 技术选型
- 纯原生 HTML + CSS + JavaScript（单页，无框架依赖）
- CSS Grid + Flexbox 布局
- CSS 变量管理深色主题色板
- 使用 Unsplash 真实图片（通过 Image CDN 参数控制尺寸）
- 本地存储（localStorage）持久化购物车数据
