## 1. 行业平均年里程标准

| 使用类型 | 年里程范围 | 说明 |
|---------|-----------|------|
| **标准通勤** | 12,000 - 15,000 英里/年 | 美国行业基准（FHWA数据） |
| **低里程** | < 10,000 英里/年 | 通常被视为"低里程"，能保值溢价 |
| **高里程** | > 15,000 英里/年 | 加速贬值阈值 |
| **极少使用** | < 5,000 英里/年 | 收藏品/备用车级别 |
| **高频使用** | > 20,000 英里/年 | 显著折价 |
| **中国标准** | 15,000 - 20,000 公里/年 | 中国私家车平均（约1.5-2万公里） |

- 10,000 英里 ≈ 16,090 公里 ≈ **理想年均值**
- 15,000 英里 ≈ 24,135 公里 ≈ **高里程分界线**

## 2. 里程对车辆价值的影响曲线模型

### 2.1 总里程与价值衰减关系（通用曲线）

里程段（英里） | 相对新车价值 | 累计贬值率 | 边际贬值率（每英里）
--------------|------------|-----------|-----------------
0 - 12,000 | 75% - 85% | 15% - 25% | 约 $0.15 - $0.20/英里
12,000 - 36,000 | 55% - 75% | 25% - 45% | 约 $0.10 - $0.15/英里
36,000 - 60,000 | 40% - 55% | 45% - 60% | 约 $0.08 - $0.12/英里
60,000 - 90,000 | 25% - 40% | 60% - 75% | 约 $0.05 - $0.08/英里
90,000 - 120,000 | 15% - 25% | 75% - 85% | 约 $0.03 - $0.05/英里
120,000+ | 5% - 15% | 85% - 95% | 约 $0.01 - $0.03/英里

### 2.2 关键拐点分析

```
价值保留率(%)
100% |💎新车
     |
 80% |    \
     |      \
 60% |       \_____ 每年约15-25%首年折旧
     |              \
 40% |               \______ 36,000英里后贬值趋缓
     |                       \
 20% |                         \______ 100,000英里后弹性很低
     |                                  \
  0% |___________________________________\____ 里程
    0   20k  40k  60k  80k  100k 120k 140k
```

**三个关键拐点：**
1. **36,000 英里（~3年）**：首次大保养节点，保修到期，价值下降加速终结
2. **60,000 英里（~5年）**：主要部件开始需要关注，价值曲线从陡降转为平缓
3. **100,000 英里（~8年）**：心理门槛，超过此值残值极低，贬值弹性极小

## 3. 溢价/折价对照表（基于KBB/Edmunds行业经验数据）

### 3.1 与"平均里程"比较的溢价折价

```text
里程低于均值     → 溢价
  -20% 里程      → +5% ~ +8% 价值
  -40% 里程      → +10% ~ +15% 价值
  -60% 里程      → +15% ~ +25% 价值

里程高于均值     → 折价
  +20% 里程      → -5% ~ -8% 价值
  +40% 里程      → -10% ~ -15% 价值
  +60% 里程      → -15% ~ -25% 价值
  +100% 里程     → -25% ~ -40% 价值
```

### 3.2 年里程溢价矩阵

| 车龄 | <5,000英里/年 | 10,000-12,000 | 15,000+ | 20,000+ |
|------|--------------|--------------|--------|--------|
| 1年 | +15~20% | 基准价 | -8~-12% | -15~-22% |
| 3年 | +10~15% | 基准价 | -10~-15% | -20~-28% |
| 5年 | +8~12% | 基准价 | -8~-12% | -15~-25% |
| 7年+ | +5~8% | 基准价 | -5~-10% | -10~-18% |

*注：随车龄增加，里程差异的影响逐渐缩小，因为年限折旧成为主导因素。*

## 4. 合理里程区间的判定标准

### 4.1 "合理"的定义按目的分类

**购买方视角（合理 = 低风险）**
- 最佳：8,000 - 12,000 英里/年（年均）
- 可接受：12,000 - 15,000 英里/年
- 超限：> 15,000 英里/年 或 总里程 > 120,000

**出售方视角（合理 = 最大化残值）**
- 理想：控制在年均 10,000 英里以下
- 可接受：控制在年均 12,000 - 14,000 英里
- 避免：超过 18,000+ 英里/年

**评估师视角（合理 = 正常使用痕迹）**
- 年里程 = 12,000 ± 3,000 英里 视为正常范围
- 超出此范围即需要做折价/溢价调整

### 4.2 里程合理性公式

```text
预期里程 = 车龄(年) × 12,000 (美国标准)
         车龄(年) × 18,000 (中国标准, 公里)

合理范围 = 预期里程 × (80% ~ 120%)
  即 9,600 ~ 14,400 英里/年（美国）
  即 14,400 ~ 21,600 公里/年（中国）

偏差影响 = (实际里程 - 预期里程) / 12,000 × 5%
  即每偏离年均12,000英里 10%, 价值调整约5%
```

## 6. 实用决策工具：里程价值调整系数

```python
def mileage_adjustment_factor(actual_miles, vehicle_age_years, market='us'):
    """
    计算里程偏差带来的价值调整系数
    返回: 0.0 ~ 1.0 的系数, 1.0 = 无调整
    """
    if market == 'us':
        expected_annual = 12000
    elif market == 'china':
        expected_annual = 18000  # 公里
    else:
        expected_annual = 12000

    expected_total = expected_annual * vehicle_age_years

    # 非线性调整: 偏差越大, 边际影响递减
    ratio = actual_miles / expected_total if expected_total > 0 else 1.0

    if ratio < 0.5:
        # 极低里程: 溢价但递减
        return min(1.25, 1.0 + (0.5 - ratio) * 0.3)
    elif ratio < 0.8:
        # 偏低里程: 线性溢价
        return 1.0 + (0.8 - ratio) * 0.15
    elif ratio < 1.2:
        # 正常范围: 微小调整
        return 1.0 - (ratio - 0.8) * 0.05
    elif ratio < 1.5:
        # 偏高: 线性折价
        return 1.0 - (ratio - 1.2) * 0.15
    else:
        # 极高: 折价递减
        return max(0.6, 1.0 - (ratio - 1.5) * 0.12)
```
