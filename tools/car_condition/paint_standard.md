# 原漆筛选标准（可接受补漆范围/面积/部位限制）

## 1. 总则

本标准用于判断车辆漆面状态是否属于「原版原漆」或「可接受补漆」（非结构性损伤、不影响车辆价值等级的正常修复）。超出以下范围的补漆，需结合事故车标准（`accident_standard.md`）综合判定是否为事故车修复痕迹。

### 关键概念定义

| 术语 | 定义 |
|------|------|
| **原版原漆** | 全车所有外露漆面均为出厂原漆，无任何后喷、补漆、划痕修复记录 |
| **可接受补漆** | 非结构件的局部小面积修复，未涉及切割/焊接/钣金整形，未伤及底材金属 |
| **不可接受补漆** | 涉及结构件、大面积、多次叠加、或伴随钣金整形（敲击/拉伸/腻子层＞2mm）的修复 |

---

## 2. 可接受补漆范围（分部位）

### 2.1 保险杠（前/后）

| 检测项 | 可接受标准 | 说明 |
|--------|-----------|------|
| 补漆面积 | 单面 ≤ 50%（一半以内） | 保险杠为塑料件，不锈蚀，全杠重喷也可接受 |
| 部位限制 | 无限制 | 保险杠属易损覆盖件，补漆不纳入事故判定 |
| 钣金修复 | 不可有任何钣金整形痕迹（塑料件无钣金概念） | 裂痕/穿透需更换而非补漆 |
| 色差要求 | 目视不明显（30°角侧光观察无明显色带） | 保险杠与相邻叶子板/翼子板色差在 ΔE ≤ 2.0 |

> **结论规则**：保险杠补漆在任何条件下 **不降级**，仅记录备注。

### 2.2 车门（四门 + 尾门/后备箱盖）

| 检测项 | 可接受标准 | 说明 |
|--------|-----------|------|
| 补漆面积 | 单面 ≤ 30% | 超出需警惕是否由碰撞引起 |
| 补漆层数 | ≤ 1 层（非多次叠加喷涂） | 多次喷涂腻子层偏厚，用漆膜仪检测（＞200μm 可疑） |
| 腻子厚度 | ≤ 200μm（含原漆层） | 超过说明有钣金整形修复 |
| 钣金修复 | 仅允许无钣金的划痕/剐蹭补漆 | 任何敲击、拉伸、焊接修复 → 不可接受 |
| 部位限制 | 允许：门板中部/下部、门把手周边 | 禁止：门框内侧（B柱对应区域）、门铰链处 |
| 数量限制 | 全车最多 2 个门有补漆 | 多于 2 个门 → 需排查是否为全车喷漆或事故 |

> **结论规则**：≤1 个门补漆（无钣金）→ **不降级**，记录备注；2 个门补漆 → 综合判断；＞2 个门或涉及任何钣金 → **降级**。

### 2.3 翼子板（前/后 + 后叶子板）

| 检测项 | 可接受标准 | 说明 |
|--------|-----------|------|
| 前翼子板 补漆面积 | ≤ 20% | 前翼子板靠近纵梁，过大面积需排查纵梁损伤 |
| 后叶子板 补漆面积 | ≤ 15% | 后叶子板与 C 柱一体，补漆面积过大说明后方碰撞严重 |
| 腻子厚度 | ≤ 150μm（含原漆层） | 后叶子板允许更薄腻子标准 |
| 钣金修复 | 仅允许无钣金的划痕补漆 | 任何切割/焊接直接影响车身结构 → 事故车标准 |
| 部位限制 | 允许：翼子板边缘/轮眉上方划痕 | 禁止：靠 A/B/C 柱连接处的补漆（伪装结构性修复） |

> **结论规则**：前翼子板少量补漆（≤20%，无钣金）→ **不降级**；后叶子板补漆（无论面积）→ 记录备注，配合事故项排查后方碰撞。

### 2.4 发动机盖（引擎盖）

| 检测项 | 可接受标准 | 说明 |
|--------|-----------|------|
| 补漆面积 | ≤ 20% | 引擎盖面积大，小范围石子崩漆/划痕补漆常见 |
| 腻子厚度 | ≤ 150μm（含原漆层） | 引擎盖高温区域，腻子过厚易开裂 |
| 钣金修复 | 仅允许无钣金的划痕修复 | 引擎盖钣金提示前方碰撞力度较大 |
| 部位限制 | 允许：前端/中间小范围 | 禁止：靠近铰链处、锁扣处、发动机舱内侧有漆雾 |

> **结论规则**：小面积（≤20%，无钣金）→ **不降级**，记录备注。

### 2.5 车顶

| 检测项 | 可接受标准 | 说明 |
|--------|-----------|------|
| 补漆面积 | ≤ 10%（且仅限冰雹坑/树胶/划痕修复） | 车顶极少单独补漆，大面积补漆疑为翻新或事故 |
| 钣金修复 | 禁止任何钣金修复 | 车顶与侧围一体，钣金即切割结构损伤 |
| 切割焊接 | 绝对禁止 | 车顶切割即「重大事故车」 |

> **结论规则**：车顶补漆（即使极小面积）也需严格记录和追问原因，任何钣金/切割 → **重大事故**。

## 3. 全车喷漆判定

| 检测项 | 判定标准 | 结论 |
|--------|---------|------|
| 全车外观统一色差异于出厂颜色代码 | 车门/叶子板内侧标签颜色代码 vs 外观实测 | 「全车改色喷漆」 |
| 车门/叶子板内侧密封条处、门铰链处有漆雾残留 | 拆开密封条可见原漆与新漆交界 | 「非原漆」 |
| 发动机舱内/后备箱内侧有油漆飞溅或色差 | 后喷漆工艺不佳导致内侧沾漆 | 「全车重喷」 |
| 全车多数（≥6个）面漆膜仪读数一致偏高（＞180μm） | 各面板厚度异常均匀 | 「全车重喷」 |
| 油箱盖内侧打开对比有明显色差 | 油箱盖外侧喷漆覆盖而内侧原漆 | 「全车喷漆」 |

> **结论规则**：全车喷漆 → 无论原因，直接 **降级**，按实际喷漆原因（改色/事故修复/翻新）综合判定。

## 4. 可接受补漆综合规则（流程图）

```
输入车辆漆面检测数据
│
├─ 扫描全车喷漆信号（色差代码/漆雾残留/均匀高读数/油箱盖色差）
│  └─ 命中任意一项 → 输出: 全车喷漆 → 降级
│
├─ 逐个面板扫描违规修复
│  ├─ 结构件（A/B/C/D柱、纵梁、底板、前围板）有补漆或钣金
│  │  └─ 命中 → 输出: 事故车修复（参考 accident_standard.md）
│  │
│  ├─ 非结构件有切割/焊接/钣金拉伸
│  │  └─ 命中 → 标记事故嫌疑
│  │
│  └─ 非结构件仅补漆（无钣金、无切割）
│     ├─ 面积 ≤ 允许上限 → 记录备注（可接受补漆）
│     └─ 面积 > 允许上限 → 标记可疑（需排查事故修复）
│
└─ 汇总结论
   ├─ 全车原漆 → 「原版原漆」
   ├─ 仅可接受补漆（逐项在允许范围内）→ 「原版原漆（可接受补漆）」★
   ├─ 超出范围但非事故 → 「非原漆（需具体情况判断）」
   └─ 涉及事故标准 → 输出对应事故结论
```

**★ 评价规则**：「原版原漆（可接受补漆）」视为 **非事故车辆的有瑕原漆**，在交易建议中如实描述：「有 XX 处小范围补漆，非事故修复，不影响结构安全。」

## 5. 边界规则与常见争议处理

| 情景 | 判定指引 |
|------|---------|
| 4S 店出厂前修复（PDI 处理） | 视为原漆状态，不降级，需有记录凭证 |
| 石子崩漆自行点漆修复（笔补/指甲油） | 视为正常磨损，不降级，记录备注（点状补漆，≤3mm） |
| 冰雹修复（无钣金、无切割、仅凹陷修复后补漆） | 车顶 ≤10%、引擎盖/后备箱 ≤15%、其他 ≤20%，不降级 |
| 划痕修复后见底漆但无金属裸露 | 不降级（清漆层修复，不属结构损伤） |
| 漆膜仪读数 >300μm 但无钣金（多次喷漆叠加） | 标记「疑似多次喷漆」，需结合其他面板综合分析 |
| 同一部位补漆后再喷漆（双层补漆） | 该部位面积按首次补漆计，记录双层补漆备注 |
| 车门铰链处或门框内侧被带漆（喷漆时遮蔽不足） | 即使面积极小，标记「喷漆工艺瑕疵」，不降级但需记录 |
| 改色贴膜车辆（揭膜后原漆完好） | 不视为补漆，贴膜残胶清理后检查原漆状态 |
| 保险杠单独更换（颜色不符需要后喷） | 记录为「更换后杠并喷漆」，不影响结构判定 |

## 6. 检查工具与操作规范

| 工具 | 用途 | 操作要点 |
|------|------|---------|
| 漆膜仪 | 测量漆面厚度，判断有无重新喷漆 | 每面至少取 3 点（中心 + 两侧边缘），取均值 |
| 色差仪（可选） | 定量测量色差 ΔE | ΔE ≤ 1.5 优秀，1.5~3.0 合格，＞3.0 明显色差 |
| 强光手电（30°~45°侧光） | 检查橘皮纹、流挂、砂纸痕等喷漆工艺 | 45°斜光照面板表面，观察橘皮纹理一致性 |
| 磁铁（磁力检测） | 快速判断腻子层厚度 | 磁力弱 → 腻子厚，磁力均匀 → 漆面薄 |
| 膜厚标准块 | 校准漆膜仪 | 使用前在已知厚度的标准块上校准 |
| 放大镜（10×~20×） | 检查边缘飞漆、砂纸痕 | 放大观察边缘交界处有无砂纸纹路 |

### 漆膜仪读数参考（以日系/德系常见车系为例）

| 部位 | 原厂厚度范围 | 补漆警戒线 | 钣金修复线索 |
|------|-------------|-----------|-------------|
| 引擎盖 | 90~130μm | ＞180μm | ＞250μm + 局部极高值（＞400μm） |
| 车门 | 80~130μm | ＞170μm | ＞220μm |
| 翼子板 | 80~140μm | ＞175μm | ＞230μm |
| 后叶子板 | 85~140μm | ＞180μm | ＞250μm |
| 车顶 | 90~140μm | ＞185μm | ＞260μm |
| 保险杠（塑料件） | 不可用漆膜仪（读数不稳定） | 目视检查色差 | 不可用磁铁法 |

> 注：以上数值为常见车型参考值，美系车/国产车原漆偏厚（+20~40μm），德系豪华车（BBA）原漆偏薄（-10~20μm）。实际检测时应以同侧对称面板的读数差值作为主要判断依据，而非绝对数值。

## 7. 术语解释

| 术语 | 解释 |
|------|------|
| **补漆** | 在保留原漆的基础上进行局部漆面修复。覆盖原有清漆层或色漆层，重新打磨-喷涂-烘干。 |
| **全车重喷（全车喷漆）** | 所有外露面板完全重新喷涂。通常涉及打磨至底材或原漆层后重新喷涂 4~6 层面漆+清漆。 |
| **原版原漆（含可接受补漆）** | 车身主体面板仍为原厂漆，仅少量局部修复且未涉及结构件、未进行钣金整形。 |
| **漆膜仪读数** | 使用涡流/磁性原理测量涂装总厚度（含电泳层+底漆+色漆+清漆）。单位：微米（μm）。 |
| **ΔE（色差值）** | 国际标准色差单位。ΔE＜1.0 肉眼不可辨；1.0~2.0 专业眼光可辨；2.0~3.0 一般目视可见；＞3.0 明显色差。 |
| **钣金整形** | 对金属面板进行敲击、拉伸、焊接填补等操作恢复形状。伴随腻子层增厚（一般＞250μm）。 |
| **橘皮纹** | 漆面干燥后表面纹理像橘子皮。原厂漆有均匀的微小橘皮；后喷漆橘皮粗细不均或完全无橘皮（过度抛光）。 |
| **漆雾** | 喷漆时喷枪飞溅到周边未遮盖部分的小颗粒漆滴。常见于发动机舱内侧、密封条内、铰链处。 |
