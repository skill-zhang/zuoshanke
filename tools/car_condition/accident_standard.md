# 事故车筛选标准

## 1. 重大事故（Structural / Frame Damage）

### 车身结构件损伤
| 检测项 | 阈值 / 判定标准 | 说明 |
|--------|----------------|------|
| 纵梁（前后） | 任何变形、褶皱、焊接、切割、修复痕迹 | 纵梁是车身主承力结构，不可修复 |
| 横梁 | 任何变形、焊接、切割、更换 | 同上 |
| A/B/C/D 柱 | 任何切割、拼接、严重钣金修复（＞3mm 变形） | 原厂焊点消失或异常为新伤 |
| 底板 | 任何切割、拼接、褶皱贯穿性变形 | 底板过孔观察梁头 |
| 前围板（防火墙） | 任何焊接、切割、穿透性损伤 | 通常伴随纵梁前段受损 |

### 安全结构件
| 检测项 | 阈值 / 判定标准 | 说明 |
|--------|----------------|------|
| 安全气囊 | 缺失、弹开/修复痕迹、饰板标签不符 | 查看仪表台/方向盘标签日期 |
| 安全带 | 锁死、更换年份不符、预紧器触发（拉出抽紧不归位） | 与车辆出厂日期差异＞2年需警惕 |
| 前防撞梁 | 任何变形、更换、修复痕迹 | 查看梁体标签是否原厂 |

### 切割与焊接
| 检测项 | 阈值 / 判定标准 | 说明 |
|--------|----------------|------|
| 车顶 | 任何切割、拼接 | 车顶与侧围一体结构，切割即重大事故 |
| 后备箱围板（后围板） | 任何切割、焊接、严重褶皱变形 | 追尾常见损伤点 |
| 后叶子板 | 任何切割、拼接 | 与 C 柱相连，切割影响结构 |

> **综合判定**：以上任意一项命中 → 标记为「重大事故车」

## 2. 泡水车（Water Damage / Flood Car）

### 一级检测（内饰水位线）
| 检测项 | 阈值 / 判定标准 | 说明 |
|--------|----------------|------|
| 座椅滑轨及底部 | 大面积锈蚀、泥沙沉积、水渍残留（非浮锈） | 正常轻微浮锈可接受 |
| 安全带全部拉出 | 霉斑、水渍痕迹、颜色分层 | 泡水后水位线以下变色严重 |
| 地板隔音棉/地毯 | 潮湿、发霉、泥沙结块、异味（霉味/腥味） | 掀开地毯检查 |
| 转向管柱/刹车踏板支架 | 大面积锈蚀（＞50% 表面积） | 通过踏板下方查看 |
| 仪表台内部线束 | 泥沙残留、接头腐蚀、异常接线修复 | 拆仪表台侧板观察 |
| OBD 接口针脚 | 锈蚀、腐蚀、氧化 | 低成本快速筛查点 |

### 二级检测（底盘与电气）
| 检测项 | 阈值 / 判定标准 | 说明 |
|--------|----------------|------|
| 保险丝盒/继电器盒 | 泥沙痕迹、水渍、锈蚀 | 发动机舱和驾驶室两处都查 |
| 车载电脑/ECU | 外壳水渍、接头腐蚀、更换标识 | 高位 ECU 可判断水位深度 |
| 车内电器功能 | 3个及以上电器同时失效（CD/车窗/空调/灯光） | 排查泡水导致线路短路 |
| 底盘 | 大面积锈蚀与泥沙沉积 | 管线缝隙中残留泥沙 |
| 发动机舱线束 | 老化异常快、泥沙残留、插头腐蚀 | 对比同年份正常车辆 |
| 后备箱备胎坑/随车工具 | 积水、锈蚀、泥沙、水渍线 | 后部泡水易积水于此 |
| 点烟器/杯架缝隙 | 泥沙、锈迹、水垢 | 低成本快速筛查点 |

### 三级检测（VIN 与历史）
| 检测项 | 阈值 / 判定标准 | 说明 |
|--------|----------------|------|
| 全车玻璃日期 | 任何一块与出厂日期差异＞6个月 | 泡水后可能全车拆解重装玻璃 |
| 仪表盘更换 | 里程数与记录不符、仪表台拆装痕迹 | 泡水常替换全车仪表台 |

> **综合判定**：一级检出 ≥3 项 + 二级检出 ≥2 项 → 标记为「泡水车」
> 三级检测用于辅助验证，不作为强制门槛

## 3. 火烧车（Fire Damage / Burned Car）

### 外观检测
| 检测项 | 阈值 / 判定标准 | 说明 |
|--------|----------------|------|
| 漆面色差 | 单面或局部过火区域漆面龟裂、起泡、变色 | 高温烘烤后漆面变化 |
| 车身塑料件（大灯、尾灯、饰条） | 熔化、变形、老化异常 | 塑料件过火后不可逆变形 |
| 前/后保险杠 | 熔化、变形、更换（无原厂日期标签） | 过火后极大概率更换 |
| 密封条 | 熔化、炭化、变形 | 车门/车窗/引擎盖密封条 |

### 发动机舱检测
| 检测项 | 阈值 / 判定标准 | 说明 |
|--------|----------------|------|
| 线束及管路 | 熔化、炭化、更换痕迹、异常接线 | 过火线束绝缘层熔毁 |
| 防火墙/隔音棉 | 过火区域明显炭化、熔化、焦糊 | 常伴随发动机舱起火 |
| 发电机/起动机/其他非易损件 | 更换或表面过火痕迹 | 正常使用极少同时更换多项 |

### 驾驶舱检测
| 检测项 | 阈值 / 判定标准 | 说明 |
|--------|----------------|------|
| 仪表台/中控台 | 熔化、变形、炭化、更换 | 驾驶舱起火常见损毁件 |
| 顶棚饰板 | 熔化、塌陷、烟熏变色 | 火烧后顶部烟熏严重 |
| 座椅 | 过火炭化、熔化、更换非原厂 | 对比座椅底部标签日期 |
| 地毯 | 炭化、过火残余 | 掀开检查有无过火痕迹 |

### 气味与残余物
| 检测项 | 阈值 / 判定标准 | 说明 |
|--------|----------------|------|
| 焦糊味 | 严重且持续（不因通风消除） | 车内长期残留过火气味 |
| 灭火剂残留（干粉/泡沫） | 白色粉末或泡沫残留物 | 扑救后清理不彻底 |
| 玻璃熔化痕迹 | 玻璃边缘高温熔化、气泡 | 火烧高温特征 |

### 底盘检测
| 检测项 | 阈值 / 判定标准 | 说明 |
|--------|----------------|------|
| 底盘隔热板/护板 | 炭化、熔化、变形 | 底盘过火特征 |
| 排气管周边车漆/密封 | 过火烘烤变色、熔化 | 排气管自身高温不属火烧 |

> **综合判定**：外观 ≥2 项 + 发动机舱 ≥2 项 + 驾驶舱 ≥1 项 → 标记为「火烧车」
> 气味与残余物、底盘检测作为辅助验证

## 4. 判定流程总则

```
输入车辆检测数据
├─ 扫描重大事故项（车身结构 + 安全结构 + 切割焊接）
│  └─ 命中任意一项 → 输出: 重大事故车
├─ 扫描泡水项（一级 ≥3 项 + 二级 ≥2 项）
│  └─ 满足 → 输出: 泡水车
├─ 扫描火烧项（外观 ≥2 + 机舱 ≥2 + 驾驶舱 ≥1）
│  └─ 满足 → 输出: 火烧车
├─ 无上述命中 → 输出: 正常车（非事故、泡水、火烧）
└─ 多类型可同时标记（如先泡后烧）
```
