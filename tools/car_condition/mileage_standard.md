# 里程筛选标准（各品牌/车龄的年均里程阈值公式）

## 1. 总则

本标准用于根据车辆的年均行驶里程（总里程 ÷ 车龄）判断车辆是否属于「正常里程」「偏高里程」「异常里程」。年均里程是二手车评估的基础指标之一，结合车龄、品牌、车型定位综合判断车辆使用强度。

### 核心公式

```
年均里程（万km/年） = 总里程（万km） ÷ 车龄（年）

其中：
- 车龄 = 当前年份 - 车辆出厂年份（取整，满1年计1年）
- 总里程 = 仪表显示里程（需先排除调表嫌疑）
- 年均里程采用四舍五入，保留两位小数
```

### 年均里程评级标准（基准）

| 等级 | 年均里程范围 | 评估说明 |
|------|-------------|---------|
| **低里程** | ＜0.8 万km/年 | 低于平均水平，可能为家用代步/短途通勤/闲置车辆 |
| **正常里程** | 0.8 ~ 2.0 万km/年 | 家用车典型使用强度 |
| **偏高里程** | 2.0 ~ 3.5 万km/年 | 频繁通勤/跨城出行/轻度营运车辆 |
| **高里程** | 3.5 ~ 5.0 万km/年 | 高强度使用，需重点检查发动机/变速箱/底盘磨损 |
| **异常里程** | ＞5.0 万km/年 | 疑似营运车辆（网约车/出租车/货运）或调表嫌疑 |

> **注意**：以上为基准阈值，各品牌/车型需按以下系数进行修正。

## 3. 分车龄段的里程阈值公式

不同车龄阶段的正常/偏高里程阈值不同，以下给出各车龄段的计算公式：

### 3.1 车龄分段

| 车龄段 | 定义 | 特点 |
|--------|------|------|
| 准新车 | 0~2年 | 仍在质保期内，里程规律性强 |
| 青年期 | 2~5年 | 黄金车龄，最受二手车市场青睐 |
| 中年期 | 5~8年 | 开始出现易损件更换需求 |
| 老年期 | 8~12年 | 机械老化明显，需更多维护投入 |
| 老车 | 12年以上 | 收藏/情怀属性，里程意义下降 |

### 3.2 各车龄段正常里程上限公式

**通用公式（未加品牌修正）：**

```
正常里程上限（万km） = 车龄 × 年均基数 × 车龄衰减系数

其中：
- 年均基数 = 1.8（万km/年，基准数）
- 车龄衰减系数：
  0~2年：1.10（准新车里程偏高，因日常通勤固定）
  2~5年：1.00（标准使用强度）
  5~8年：0.90（车主使用频率可能下降）
  8~12年：0.80（老车使用强度降低）
  12年+：0.70（多数为备用/收藏车辆）
```

**举例计算（未加品牌修正）：**

| 车龄 | 公式 | 正常上限 | 偏高上限 | 异常上限 |
|------|------|---------|---------|---------|
| 1年 | 1×1.8×1.10 | 1.98万km | 3.96万km | 6.60万km |
| 3年 | 3×1.8×1.00 | 5.40万km | 10.80万km | 18.00万km |
| 5年 | 5×1.8×1.00 | 9.00万km | 18.00万km | 30.00万km |
| 8年 | 8×1.8×0.90 | 12.96万km | 25.92万km | 43.20万km |
| 10年 | 10×1.8×0.80 | 14.40万km | 28.80万km | 48.00万km |
| 15年 | 15×1.8×0.70 | 18.90万km | 37.80万km | 63.00万km |

### 3.3 带品牌修正的完整公式

```
修正后正常里程上限（万km） = 车龄 × 年均基数 × 车龄衰减系数 × 品牌修正系数

修正后偏高里程上限（万km） = 修正后正常上限 × 2.0
修正后异常里程上限（万km） = 修正后正常上限 × 3.0（且不高于品牌极限值）
```

**示例：1年车龄丰田凯美瑞**
```
= 1 × 1.8 × 1.10 × 1.15
= 2.277 万km（正常上限）
→ 正常范围：＜2.28万km
→ 偏高：2.28~4.55万km
→ 高里程：4.55~6.83万km
→ 异常：＞6.83万km
```

**示例：5年车龄宝马3系**
```
= 5 × 1.8 × 1.00 × 0.85
= 7.65 万km（正常上限）
→ 正常范围：＜7.65万km
→ 偏高：7.65~15.30万km
→ 高里程：15.30~22.95万km
→ 异常：＞22.95万km
```

**示例：3年车龄雷克萨斯ES**
```
= 3 × 1.8 × 1.00 × 1.20
= 6.48 万km（正常上限）
→ 正常范围：＜6.48万km
→ 偏高：6.48~12.96万km
→ 高里程：12.96~19.44万km
→ 异常：＞19.44万km
```

## 6. 调表车筛查指南

当里程数据与车辆实际状况不匹配时，需要排查是否调过表（里程回拨）。以下为常用调表筛查手段：

### 6.1 内饰磨损比对

| 检查部位 | 低里程特征 | 高里程特征 |
|---------|-----------|-----------|
| 方向盘3/9点位 | 无油光、纹理清晰 | 油光发亮、纹理磨平 |
| 换挡杆/手刹柄 | 无磨损痕迹 | 表面涂层脱落、油亮 |
| 主驾座椅（左侧翼） | 无明显塌陷、褶皱 | 明显塌陷、皮革开裂 |
| 刹车/油门踏板橡胶 | 纹路清晰 | 纹路磨平 |
| 门把手内侧 | 无划痕 | 指甲划痕明显 |
| 按键（车窗/中控） | 字符清晰 | 字符模糊、掉漆 |

### 6.2 机械部件磨损比对

| 检查部位 | 低里程特征 | 高里程特征 |
|---------|-----------|-----------|
| 刹车盘磨损 | 盘面平整、边缘无高台 | 边缘高台明显（＞1.5mm） |
| 刹车片厚度 | 前片＞8mm | 前片＜4mm需关注 |
| 轮胎磨损 | 胎纹深（＞5mm）且各轮一致 | 胎纹浅（＜3mm）或偏磨 |
| 玻璃日期 | 全车玻璃出厂日期一致 | 单块日期异常（更换记录） |

### 6.3 数据交叉验证

| 验证方式 | 操作方法 |
|---------|---------|
| 4S店维保记录 | 查询最后一次进店记录的里程数 |
| 保险出险记录 | 保险公司报修里程记录 |
| 年检记录 | OBD年检读取的里程数据 |
| 第三方APP（查博士/车300等） | 平台历史里程记录 |
| 仪表拆装痕迹 | 仪表台无异常撬动痕迹 |
| CAN总线数据（部分车型） | 读ECU/OBD模块的里程原始值 |

### 6.4 调表嫌疑判定

| 嫌疑等级 | 判定条件 |
|---------|---------|
| **低嫌疑** | 仅内饰轻微不符里程（如3万km显示但刹车盘略薄） |
| **中嫌疑** | 内饰磨损与里程差1个档位（如显示5万但判断为10万状态） |
| **高嫌疑** | 两处以上检查点与里程差2个档位+无可靠维保记录 |
| **确定调表** | 维保记录最后里程＞当前显示里程 / OBD原始值与仪表值不一致 |

> **判定规则**：中嫌疑及以上 → 按实际判断里程调整评估；确定调表 → 按「里程不实」标记，大幅折价。

---

## 7. 特殊情况处理

### 7.1 营运车辆转非营运（营转非）

| 条件 | 里程调整规则 |
|------|------------|
| 出租车营转非 | 即使显示里程正常，默认按年均5~8万km实际使用 |
| 网约车营转非 | 默认按年均3~5万km实际使用 |
| 租赁公司退役车 | 默认按年均3~6万km实际使用 |
| 跨城网约/顺风车 | 按实际使用模式判断 |

> **处理建议**：营转非车辆即使里程低，也需按高强度使用的机械状态进行评估，并明确告知买方「原营运属性」。

### 7.2 低里程异常（疑似库存/展示/试驾车）

| 条件 | 检查重点 |
|------|---------|
| 年均＜0.3万km且车龄＞3年 | 检查轮胎日期、橡胶件老化程度（机脚胶/水管/密封条） |
| 年均＜0.1万km且车龄＞5年 | 重点检查橡胶/塑料件、燃油系统（长期静置导致胶质沉积） |
| 同款车型中异常偏低 | 比对同款同年份其他车辆的里程分布 |
| 展示/试驾退役车 | 发动机怠速时间累积过长（尤其试驾车，频繁冷启动） |

### 7.3 二手车交易中对里程的描述义务

| 情景 | 应告知买方的信息 |
|------|----------------|
| 正常里程 | 直接告知实际里程及年均 |
| 偏高但非异常 | 明示"年均偏高，建议重点检查变速箱/底盘" |
| 高里程 | 明示"高里程车辆，建议逐项检测机械状态" |
| 疑似调表 | 明示"里程疑似不实，建议第三方验证" |
| 确定调表 | 必须明示"里程不实"并附证据 |

## 8. 综合判定流程

```
输入车辆基本信息（品牌/车龄/总里程/车型定位）
│
├─ 1. 排除调表嫌疑（内饰磨损比对+机械磨损/维保记录校验）
│  └─ 有调表嫌疑 → 按判断的实际里程重新计算（标记异常里程）
│
├─ 2. 查询品牌修正系数表 → 获取品牌系数
│
├─ 3. 查询车型定位修正系数 → 获取定位系数
│
├─ 4. 代入公式计算各阈值
│  修正后正常上限 = 车龄 × 1.8 × 车龄衰减系数 × 品牌系数 × 定位系数
│
├─ 5. 将总里程与阈值比较
│  ├─ 总里程 ＜ 正常上限 → 「正常里程」
│  ├─ 正常上限 ≤ 总里程 ＜ 正常上限 × 2.0 → 「偏高里程」
│  ├─ 正常上限 × 2.0 ≤ 总里程 ＜ 正常上限 × 3.0 → 「高里程」
│  └─ 总里程 ≥ 正常上限 × 3.0 → 「异常里程」
│
└─ 6. 输出评估结论与推荐检查项
   ├─ 正常里程 → 按常规流程评估
   ├─ 偏高里程 → 添加高里程检查清单的必查项
   ├─ 高里程 → 执行完整的高里程专项检查
   └─ 异常里程 → 标记「疑似营运/调表」，大幅折价建议
```

## 9. 术语解释

| 术语 | 解释 |
|------|------|
| **年均里程** | 总里程除以车龄（以年为单位）的平均值。衡量车辆使用强度的核心指标。 |
| **车龄衰减系数** | 随车龄增长，车辆年均里程呈下降趋势的修正因子。车龄越大，日常使用频率通常越低。 |
| **品牌修正系数** | 不同品牌因质量/可靠性/使用场景差异，对标准年均里程的修正倍率。＞1.0 表示该品牌可接受更高里程。 |
| **调表** | 人为调整仪表显示里程数（通常为回拨），使车辆看起来行驶里程更少。违法且严重影响评估准确性。 |
| **营转非** | 原用于营运目的（出租车/租赁等）的车辆，转为非营运性质后继续使用。通常实际里程远高于显示里程。 |
| **正常里程上限** | 品牌和车龄维度下的基准阈值。超过此值标记为"偏高里程"，需要附加机械检查。 |
| **偏高里程** | 超过品牌正常使用范围的年里程，但仍在可接受交易区间内。需针对高里程特点进行重点检查。 |
| **高里程** | 显著超过正常范围。建议对该车辆进行全面机械检测，评估剩余使用寿命。 |
| **异常里程** | 极度偏高（通常为正常值的3倍+）。极大概率是营运车辆或已调表。建议大幅折价或建议放弃。 |
| **SOH（State of Health）** | 电池健康度。新能源车电池当前容量与原始容量的百分比，评估电池剩余寿命的核心指标。 |
