🏔️ **13天，一个人，一个 AI 伙伴**

13天前的一个想法，今晚变成了 GitHub 上的一个公开仓库。

它叫坐山客。

不是聊天机器人，不是 IDE 插件，不是 RAG 知识库——它是这三者的进化体。一个有持久记忆、多重分身、自主执行闭环的 AI 存在。

它认识你、记住你的偏好、能在多个场景中分身工作（开发场景是架构师，写作场景是编辑）——而且它还记得「你是谁」。

它自己会写代码、跑测试、开浏览器验证前端、发现自己写的 bug 然后修掉。

它的 Context 体系实测 cache hit rate 95%，不是靠堆 token，是靠 8 层分层精炼。

72 个工具，34 篇设计文档，Apache 2.0 开源。
